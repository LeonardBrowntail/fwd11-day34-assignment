## Description

This project is my submission for two assigments (day 34 and 38) from the Full Stack Web Development Bootcamp by <a href="https://dibimbing.id/">Dibimbing.id</a>.

## Day 34 goals

The first assignment (day 34) goals were to test how well students:

<ul>
  <li>Implement REST API,
  <li>Create CRUD (Create, Read, Update, Delete) operations for a <a href="https://github.com/LeonardBrowntail/simple-css-prototype">previous case study</a>,
  <li>Connect the API with <a href="https://github.com/LeonardBrowntail/dibimbing-assignment-day23">previously made database</a>,
  <li>Return a consistent and structured API response, and
  <li>Validate API inputs.
</ul>

## Day 38 goals

The second assignment (day 38) goals were to test how well students:

<ul>
  <li>Implement authentication on REST API using Laravel Sanctum</li>
  <li>Create and manage well structured Postman collection</li>
  <li>Use Postman's environment variables and scripting to automate API testing</li>
  <li>Test all CRUD features using the built API infrastucture</li>
</ul>
